# 📡 API Testing & Collections

## 📌 Objetivo
Repositorio de colecciones de **Postman** para la validación de los endpoints de WooCommerce que gestionan el carrito y la personalización de productos.

## 🧪 Pruebas Planificadas:
- **Validación de Esquemas:** Asegurar que los JSON de respuesta sean correctos.
- **Pruebas de Seguridad:** Intentar bypass de validaciones vía API directamente (REST API).
- **Scripts de Pre-solicitud:** Generación de payloads dinámicos para pruebas de estrés.

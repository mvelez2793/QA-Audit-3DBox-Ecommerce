# 🛡️ Ecosistema de Seguridad y Vulnerabilidades (OWASP)

## 📌 Objetivo
Este módulo centraliza las auditorías de seguridad perimetral y de aplicación realizadas sobre la plataforma 3D-BOX, bajo el estándar internacional **OWASP Top 10**.

## 🛑 Vulnerabilidades Sistémicas Detectadas
- **VULN-01 (OWASP A04:2021):** Inyección masiva de payloads en el Backend (Resource Exhaustion). *Estatus: Mitigación pendiente vía Smoke Test Automático.*
- **VULN-02 (Próximamente):** Auditoría de sanitización en inputs de búsqueda y formularios (SQLi / XSS).
